Diferente de tudo que você já viu!

O lidiun 5 foi desenvolvido em um padrão chamado de Segment. Diferente do tradicional MVC, ele permite separar o código HTML do PHP.

Com o Segment, você pode criar classes PHP chamadas de Renders, que renderizarão o layout trazendo o html para dentro do PHP em variáveis e substituindo TAGs por segments ("Blocos de html") montados com informações geralmente provindas do banco de dados.

Como usar o lidiun 5

1º Leia e entenda sobre o Segment design pattern. Go!
2º Leia e entenda a estrutura de pastas. Go!
3º Faça o download do framework no botão ao lado. Download!
4º Siga os exemplos da documentação lendo os tutorial. Go!
